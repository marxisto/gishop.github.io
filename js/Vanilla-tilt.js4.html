vanille-tilt.js

Une bibliothèque javascript d'inclinaison 3D fluide dérivée de Tilt.js (version jQuery)

  • Lisse.
  • Poids léger.
  • 60FPS.
  • VanillaJS.
  • Pas de dépendances.
  • Facile à utiliser

Comment utiliser

Moyen facile

<body>
<div class="your-element" data-tilt></div>

<!-- at the end of the body -->
<script type="text/javascript" src="vanilla-tilt.js"></script>
</body>

Plus d'options

<div class="your-element" data-tilt data-tilt-max="50" data-tilt-speed="400" data-tilt-perspective="500"></div>

Toutes les options avec les valeurs par défaut

{
    reverse:                false,  // reverse the tilt direction
    max:                    35,     // max tilt rotation (degrees)
    startX:                 0,      // the starting tilt on the X axis, in degrees.
    startY:                 0,      // the starting tilt on the Y axis, in degrees.
    perspective:            1000,   // Transform perspective, the lower the more extreme the tilt gets.
    scale:                  1,      // 2 = 200%, 1.5 = 150%, etc..
    speed:                  300,    // Speed of the enter/exit transition
    transition:             true,   // Set a transition on enter/exit.
    axis:                   null,   // What axis should be disabled. Can be X or Y.
    reset:                  true,   // If the tilt effect has to be reset on exit.
    easing:                 "cubic-bezier(.03,.98,.52,.99)",    // Easing on enter/exit.
    glare:                  false,  // if it should have a "glare" effect
    "max-glare":            1,      // the maximum "glare" opacity (1 = 100%, 0.5 = 50%)
    "glare-prerender":      false,  // false = VanillaTilt creates the glare elements for you, otherwise
                                    // you need to add .js-tilt-glare>.js-tilt-glare-inner by yourself
    "mouse-event-element":  null,   // css-selector or link to HTML-element what will be listen mouse events
    gyroscope:              true,   // Boolean to enable/disable device orientation detection,
    gyroscopeMinAngleX:     -45,    // This is the bottom limit of the device angle on X axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the left border of the element;
    gyroscopeMaxAngleX:     45,     // This is the top limit of the device angle on X axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the right border of the element;
    gyroscopeMinAngleY:     -45,    // This is the bottom limit of the device angle on Y axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the top border of the element;
    gyroscopeMaxAngleY:     45,     // This is the top limit of the device angle on Y axis, meaning that a device rotated at this angle would tilt the element as if the mouse was on the bottom border of the element;
}

Manière JS

<body>
<div class="your-element"></div>

<script type="text/javascript" src="vanilla-tilt.js"></script>
<script type="text/javascript">
	VanillaTilt.init(document.querySelector(".your-element"), {
		max: 25,
		speed: 400
	});
	
	//It also supports NodeList
	VanillaTilt.init(document.querySelectorAll(".your-element"));
</script>
</body>

L'utilisation la plus basique :

  1. Ajouter 
    <script src="vanilla-tilt.js"></script>
    juste avant la fermeture 
    </body>
    étiqueter
  2. Marquez vos éléments avec 
    <div data-tilt></div>

Créer un effet de parallaxe

  1. Ajouter 
    transform-style: preserve-3d
    à votre élément d'inclinaison.
  2. Ajouter 
    transform: perspective(1000px);
    à votre élément d'inclinaison. (1000px est la perspective par défaut, mais peut être modifiée)
  3. Ajouter un 
    transform: translateZ(20px)
    à vos éléments intérieurs qui doivent sortir.

Effet d'éblouissement

La définition de cette option activera un effet d'éblouissement. Vous pouvez modifier la valeur d'éblouissement avec 

data-tilt-max-glare

<div data-tilt data-tilt-glare data-tilt-max-glare="0.8"></div>

Inclinaison inversée

Le réglage de cette option inversera l'inclinaison.

<div data-tilt data-tilt-reverse="true"></div>

Continuez à flotter

La définition de cette option ne réinitialisera pas l'élément d'inclinaison lorsque la souris de l'utilisateur quitte l'élément.

<div data-tilt data-tilt-reset="false"></div>

Écoute pleine page

En définissant cette option, l'élément répondra à tous les mouvements de souris sur la page.

<div data-tilt data-tilt-full-page-listening></div>

Échelle au survol

Le réglage de cette option mettra à l'échelle l'élément d'inclinaison lors du survol.

<div data-tilt data-tilt-scale="1.1"></div>

Position d'inclinaison de départ

La définition de cette option inclinera l'élément à des degrés spécifiques au chargement de la page.

<div data-tilt data-tilt-startX="20" data-tilt-startY="-20"></div>

Désactiver l'axe X

Le réglage de cette option désactivera l'axe X sur l'élément d'inclinaison.

<div data-tilt data-tilt-axis="y"></div>

Désactiver l'axe Y

Le réglage de cette option désactivera l'axe Y sur l'élément d'inclinaison.

<div data-tilt data-tilt-axis="x"></div>

Détruire la méthode

Appelez cette méthode supprimera tous les événements et classes de l'élément tilt.

let destroyBox = document.querySelector("#destroy-box");
VanillaTilt.init(destroyBox);

document.querySelector("#destroy-button").addEventListener("click", function () {
    destroyBox.vanillaTilt.destroy();
});

document.querySelector("#enable-button").addEventListener("click", function () {
    VanillaTilt.init(destroyBox);
});

Événement de changement d'inclinaison

L'événement de changement d'inclinaison affichera les x, y et les pourcentages d'inclinaison.

let eventBox = document.querySelector("#box-event");
let outputContainer = document.querySelector(".output");
VanillaTilt.init(eventBox);


eventBox.addEventListener("tiltChange", function (event) {
	let li = document.createElement("li");
	li.textContent = JSON.stringify(event.detail);
	outputContainer.insertBefore(li, outputContainer.firstChild);
});
